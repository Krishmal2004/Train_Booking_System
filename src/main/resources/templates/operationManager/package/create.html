Package Creation Tips
  • Package ID should follow the format PKG-YYYY-XXX where YYYY is the year and XXX is a sequential number
  • Set the Status to "Scheduled" for future packages or "Active" for immediate availability
  • Add detailed Remarks to help team members understand the package details
  • Ensure the End Date is after the Start Date
  • Seasonal packages typically range from 1 week to 3 months in duration
